Se requiere un Nickname para jugar El nickname debe tener 3 o más caracteres El nickname no puede tener más de 15 caracteres
Se requiere un ID para jugar El ID debe tener 6 o más caracteres El ID no puede tener más de 15 caracteres

Instrucciones:

  1. Ingresa un nickname con el que tus amigos puedan identificarte
  2. Ingresa el ID de la sala. Si la sala no existe se creará!
  3. Pueden participar hasta 5 jugadores a la vez
  4. Acumula puntos encontrando los pares de cartas que sean idénticas
  5. Si encuentras un par idéntico tendrás un turno extra!
